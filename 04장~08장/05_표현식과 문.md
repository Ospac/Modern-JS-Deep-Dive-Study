# 05장 표현식과 문

용어들의 정의가 중심이 되는 장

## 5.1 값

- **값**<sup>value</sup> : 표현식이 평가되어 생성된 결과
- 평가: 식을 해석해서 값을 생성하거나 참조하는 것

```js
10 + 20; -> 해당 표현식을 평가(해석)하여 값 30을 생성한다.
```

10 + 20은 변수에 할당 이전에 평가되어 값을 생성해야 한다.

```js
var sum = 10 + 20;
```

## 5.2 리터럴

**리터럴**<sup>literal</sup>은 문자열, 숫자, 불리언, null , undefined, 객체, 배열 등으로 값을 생성하는 표기법, 혹은 데이터(값) 그 자체를 의미함.

```js
ex. 100, 10.5, 0x41, 'Hello', true, false, null, undefined,
{ name: 'Lee', address: 'Seoul' },
[ 1, 2, 3]
function() {},
/[A-Z]+/g
```

## 5.3 표현식

**표현식**<sup>expression</sup>은 값으로 평가될 수 있는 문<sup>statement</sup>이다.
표현식은 리터럴, 식별자, 연산자, 함수 호출등의 조합으로 이루어질 수 있으며, 모두 값으로 평가되는 문이다. **따라서 표현식은 하나의 값처럼 사용할 수 있다.**

#### 리터럴이 표현식일때

```js
var score = 100;
```

`100`은 리터럴이다. `100`은 평가되어 10진수 숫자 값 100을 생성하므로 그 자체로 표현식이다.

#### 리터럴과 연산자가 합쳐진 표현식

```js
var score = 50 + 50;
```

`50 + 50`은 평가되어 숫자 값 100을 생성하므로 표현식이다.

#### 식별자가 표현식

```js
var newScore = score + 50; // score + 50은 하나의 값으로 평가되어, newScore에 할당된다.
var name = person.name;
```

변수 식별자는 참조하면 변수 값으로 평가되는 표현식이다.

#### 함수/메서드 호출 표현식

```js
var name = person.getName();
```

## 5.4 문

**문**<sup>statement</sup>은 프로그램을 구성하는 기본 단위이자 최소 실행 단위. 이러한 문장들, 문의 집합으로 이루어 진 것이 프로그램이다.
ex.

- 선언문
- 할당문
- 조건문
- 반복문

## 5.5 세미콜론과 세미콜론 자동 삽입 기능

세미콜론(**;**)은 문의 종료를 의미한다. 자바스크립트 엔진은 세미콜론으로 문이 종료한 위치를 확인, 순차 실행한다.

- { ... } 중괄호 코드 블록뒤에는 붙이지 않는다. ex. if, for, 함수
- 세미콜론 자동 삽입 기능(ASI): 자바스크립트 엔진이 문의 끝이라고 예측되는 지점에 세미콜론을 자동으로 붙여주는 기능이 있다.
- 단, 개발자의 예측과는 다르게 동작하는 경우들이 있어서 유의

```js
function foo() {
  return;
  {
  }
  // ASI의 동작 결과 => return; {};
  // 개발자의 예측 => return {};
}

console.log(foo()); // undefined

var bar = (function () {})(function () {})();
// ASI의 동작 결과 => var bar = function () {}(function() {})();
// 개발자의 예측 => var bar = function () {}; (function() {})();
// TypeError: (intermediate value)(...) is not a function
```

## 5.6 표현식인 문과 표현식이 아닌 문

문은 코드의 기본단위 (5.4)
그 중에서도 **표현식**은 값으로 평가될 수 있는 문이다. 따라서, 표현식은 값으로 치환할 수 있고, 변수에 할당 가능하다면 표현식인 문이다.

```js
var x; // 변수 선언문은 값으로 평가 될 수 없음


x = 1 + 2; // 3으로 평가되는 표현식

var foo = var x // SyntaxError: Unexpected token var
var foo = x = 1 + 2; // ok!
```
